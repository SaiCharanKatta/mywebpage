HTML Form

An HTML Form is a section of the document that collects input from the user. The input from the user is generally sent to a server (Web servers, Mail clients, etc). We use the HTML <form> element to create forms in HTML.

HTML Form Elements

There are various HTML form elements for various input types. The form elements available in HTML5 are as follows:

HTML <input> tag
HTML <label> tag
HTML <button> tag
HTML <select>, <option> and <optgroup> tags
HTML <textarea> tag
HTML <fieldset> tag
HTML <legend> tag
HTML <datalist> tag
HTML <output> tag

Form tag represent HTML form, this consist of form elements.

<form> tag

Attributes:

Name/id : name of the form/name of the object
Method: Request method (HTTP Request Methods)
Action: URL (Resource to which form is submitted/data is send)
Content-type: Uploading files (MIME Type  Multi Purpose Internet Mail Extension)

HTML <input> tag

The HTML <input> tag defines the field where the user can enter data.
The <input> element in HTML is used to create form controls for user input. There are several types of <input> elements, each with a specific purpose. Here are some of the most common types of <input> elements:
type="text": This creates a text input field where the user can enter free-form text.
type="email": This creates a text input field that is specifically designed for email addresses, and may trigger a validation error if an invalid email format is entered.
type="password": This creates a text input field where the characters are hidden, such as when entering a password.
type="number": This creates a text input field that only accepts numerical input.
type="checkbox": This creates a checkbox control that allows the user to select one or more options.
type="radio": This creates a radio button control that allows the user to select a single option from a group of options.
type="submit": This creates a button that, when clicked, submits the form data to the server.
type="reset": This creates a button that, when clicked, resets the form to its initial values.
type="file": This creates a control that allows the user to select a file from their device
